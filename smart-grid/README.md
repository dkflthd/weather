# AI 스마트 그리드 시뮬레이션

학생들이 웹 브라우저에서 바로 실행할 수 있는 정적 시뮬레이션입니다.

## 파일 구성
- `index.html`
- `style.css`
- `script.js`

## 실행 방법
1. 이 파일들을 같은 폴더에 업로드합니다.
2. GitHub 저장소에 푸시합니다.
3. GitHub Pages를 켜면 웹에서 바로 실행할 수 있습니다.

## 활동 목표
- 재생에너지 변동성 이해
- 배터리 저장의 역할 이해
- 기본 배분과 AI 예측 배분 비교
- 비용, 계통전력, 탄소배출 변화 해석

## 학생 활동 예시
- 태양광 규모를 높였을 때 어떤 시간이 가장 안정적인가?
- 배터리 용량을 늘리면 Grid 사용량이 얼마나 줄어드는가?
- 겨울/여름 조건에서 AI 배분의 효과 차이는 무엇인가?
